# QuizArena 操作手冊

> 本手冊分為「學生操作手冊」與「老師操作手冊」兩部分，提供從登入到匯出報表的完整流程。

---

## 學生操作手冊

### 1. 如何登入

1. 開啟 QuizArena 首頁（`index.html`）。
2. 在登入區輸入帳號密碼，或選擇「註冊新帳號」建立新帳號。
3. 登入後，系統會把帳號寫入 `localStorage.currentUser`，後續所有頁面（含錯題本、雷達圖、個人資料）都會以這個帳號為主。

> ⚠️ 注意：玩家暱稱（playerName）只用於房間遊戲顯示，不會影響錯題本的查詢。

---

### 2. 如何加入房間

1. 在首頁點選「加入房間」。
2. 輸入 6 位數 PIN（由老師或主持人提供）。
3. 輸入暱稱、選擇角色或隊伍。
4. 等待主持人開始遊戲。

---

### 3. 如何答題

1. 房間開始遊戲後，題目會依序出現。
2. 點選選項作答（單選 / 多選 / 填充 / 配對 / 是非題）。
3. 答錯題目會自動進入「我的錯題本」。
4. 故事模式答錯也會同步到錯題本。

---

### 4. 如何查看錯題本

1. 從首頁進入「我的錯題本」（或直接前往 `wrong_book.html`）。
2. 進站時自動載入錯題清單，並顯示：
   - 錯題數
   - 累積答錯次數
   - 目前帳號
3. 若顯示「尚未登入」警示，請先回首頁登入。

---

### 5. 如何依科目篩選錯題

在「完整複習清單」上方有科目篩選 chips：

```
[全部] [國文] [英文] [數學] [自然] [社會] [程式] [其他]
```

- 點選任一科目即可只顯示該科目的錯題。
- 同時可使用「排序方式」下拉：
  - 最近錯誤
  - 答錯最多次
  - 難度高 → 低
  - 標題排序
- 篩選後右上角會顯示「（顯示 X / Y 題）」。

> 系統會用 `category` → `sourceBankTitle` → 內建科目對照表自動歸類。

---

### 6. 如何使用錯題練習

1. 在「錯題練習」區塊點「開始練習」。
2. 系統會洗牌錯題、逐題重新作答。
3. 答完後可以：
   - 看綠色正解
   - 看解析
   - 按「AI 家教講解」叫 AI 個人化解析
   - 按「下一題」繼續

---

### 7. 如何查看雷達圖

錯題本頁面有兩個雷達圖：

#### 7.1 學習弱點雷達圖
- 由本地資料計算，依「科目」統計錯題比例。
- 圖例會用顏色（紅 / 橘 / 綠）標示嚴重度。

#### 7.2 能力雷達圖
- 從後端 `/api/radar/<player>` 取得。
- 顯示三條線：
  - 紫色 = 你
  - 灰色虛線 = 全體平均
  - 金色 = 第一名
- 可以在輸入框換查詢帳號，按「重新查詢」即可比對其他玩家。
- 如果你還沒有比賽紀錄，會顯示「目前沒有比賽紀錄，完成遊戲後會產生能力分析」。

---

### 8. 如何寫錯題筆記

1. 每張錯題卡下方都有「我的筆記・記憶法」黃色欄位。
2. 輸入自己的記憶法、補充說明或聯想口訣。
3. 按「儲存筆記」即可。
4. 筆記只存在你自己的瀏覽器（`localStorage`），跨裝置不會同步。

---

### 9. 如何使用 AI 解題講解

#### 9.1 整體建議
1. 在「複習建議」區塊點「AI 給建議」。
2. AI 會分析你的錯題分布並給：
   - 學習狀況分析
   - 三個改進建議
   - 推薦複習題型
   - 建議練習順序
   - 鼓勵語

#### 9.2 單題講解
1. 在錯題卡或練習區點「AI 解題講解」。
2. AI 會針對該題說明：
   - 為什麼選項正確 / 錯誤
   - 解題技巧
   - 類似題型提醒

## 老師操作手冊

### 1. 如何建立題庫

1. 登入後從首頁進入「題庫中心」（`create_home.html`）。
2. 按「+ 新建題庫」輸入名稱。
3. 選擇遊戲模式（個人 / 隊伍）。
4. 點「+ 新增題目」填入：
   - 題目內容
   - 題目類型（單選 / 多選 / 是非 / 填充 / 配對）
   - 選項與正確答案
   - 解析
   - 分類、難度
   - 配圖（選填）
5. 系統會自動把草稿存到 `quizarena_draft_<user>` 作為瀏覽器備援，避免網路中斷時資料消失。
6. 按「儲存題庫」會同步到伺服器。

> 也可以從 AI 生成題庫：點「AI 生成」輸入主題、難度、題數，由 AI 自動產生整份題庫。

---

### 2. 如何建立房間

1. 在題庫中心選好要使用的題庫。
2. 點「建立房間」會出現房間設定 modal：
   - 是否需要密碼
   - 答題模式
   - 計分模式
3. 確認後系統會回傳 6 位數 PIN。
4. 把 PIN 給學生加入。

---

### 3. 如何查看老師報表

1. 進入「老師模式」（`teacher_report.html`）。
2. 輸入 6 位數 PIN，按「查看報表」即可載入該房間的：
   - 平均正確率、平均分數
   - 圖表分析
   - 學生表現表格
   - 題目答對率

---

### 4. 如何選擇歷史房間

頁面有兩個方式：

#### 4.1 建房紀錄列表
- 在「建房紀錄」區塊點任一場次即可載入。
- 按「重新整理」可以更新列表。

#### 4.2 進階分析儀表板房間下拉
- 在「進階分析儀表板」標題列有「房間：」下拉。
- 選好房間後，下方所有分頁（班級總覽 / 個別學生 / 弱題分析 / AI 建議 / 家長報告）都會同步更新。
- 若該房間 PIN 是 6 位數，上方主報表也會同步切換到該房間。

---

### 5. 如何查看學生個別表現

1. 在「進階分析儀表板」選「👤 個別學生」分頁。
2. 左側顯示學生列表，點任一學生即可看到：
   - 排名（第 N 名 / 共 X 人）
   - 正確率、平均速度、本場分數
   - 弱點科目（紅色標籤）
   - 各科分數進度條

---

### 6. 如何找出弱題

1. 在「進階分析儀表板」選「⚠️ 弱題分析」分頁。
2. 上方長條圖顯示前 10 題錯誤率排行。
3. 下方列出每題的：
   - 題號
   - 錯誤率
   - 答錯的學生名單

> 老師可以依此調整下一輪的教學重點。

---

### 7. 如何產生家長報告

1. 在「進階分析儀表板」選「📄 家長報告」分頁。
2. 從「選擇學生」下拉選一位學生。
3. 點「生成報告」。
4. 系統會回傳一個分享連結：
   - 可以複製連結傳給家長
   - 或直接點「預覽報告」開啟頁面
5. 家長透過連結就能看到學生表現、弱點、AI 建議。

---

### 8. 如何匯出 CSV / Excel / HTML 報表

頁面頂部有三個匯出按鈕（載入報表後才會啟用）：

| 按鈕 | 用途 |
|------|------|
| **匯出 CSV** | 純文字 CSV，方便用 Excel 開啟、做樞紐分析 |
| **匯出 Excel 報表** | 含格式的 Excel，多份工作表 |
| **匯出分析報告** | 完整 HTML 報告，可直接用瀏覽器開啟、列印或寄給家長 |

點選後會以 `quizarena-<報表類型>-<PIN>.<副檔名>` 為檔名下載。

## 常見問題

### Q1. 錯題本顯示「尚未登入」怎麼辦？
回首頁登入。錯題本一律使用帳號 `currentUser`，不會用玩家暱稱。

### Q2. 為什麼能力雷達圖空白？
- 該帳號沒有比賽紀錄 → 完成一場遊戲後就會出現。
- 後端 API 暫時無回應 → 頁面會顯示「能力雷達 API 回傳錯誤」並可重試。
- 等待 Chart.js 載入中 → 慢一點再看。

### Q3. 今日複習為什麼是空的？
- 你今天還沒有新的錯題，請先去做題。
- 如果你「剛剛」答錯一題卻沒出現：按「重新整理」按鈕。

### Q4. 草稿不見了？
- 題庫中心會自動把草稿存到 `localStorage.quizarena_draft_<user>`。
- 若伺服器斷線時建立題庫，下次進站會出現「已從瀏覽器草稿恢復題庫資料」提示。

### Q5. 我換瀏覽器筆記就消失了？
是。錯題筆記目前存在 `localStorage`，跨裝置不會同步。
